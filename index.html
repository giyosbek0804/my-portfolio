03 — Problem Solving

Technical Challenges

Real-world engineering problems I've encountered and the creative solutions I implemented to overcome them.

YouTune: Solving API Quota Exhaustion

Performance Optimization API Design Firebase Caching
The Challenge

Fetching home feeds for 100+ subscriptions using the standard YouTube Search API was consuming thousands of quota units per minute, quickly leading to 429: Too Many Requests errors and app shutdown for users.

The "UU" Hack Solution

Discovered that every channel ID (UC...) has a matching "Uploads" playlist ID (UU...). By converting the ID and using playlistItems.list instead of search, I reduced the quota cost from 100 units to 1 unit per fetch.

99% Quota Reduction
48h Firebase Cache
Instant Feed Loading
I implemented a multi-layer caching strategy: the app fetches ~120 videos once, stores them in Firestore with a 48-hour timestamp, and serves them locally. This combination of the "UU" hack and persistent caching turned a budget-breaking problem into a high-performance feature.
